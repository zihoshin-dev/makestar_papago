
# MAKESTAR Papago (K-Pop Fandom Translator)

## 📋 프로젝트 개요
메이크스타 커머스 및 K-팝 팬덤 환경에 최적화된 다국어 번역 서비스입니다. 
단순한 기계 번역을 넘어 메이크스타만의 고유 용어와 맥락을 반영한 압도적인 번역 경험을 제공합니다.

## 🛠 기술 스택
- **Frontend:** Next.js, Tailwind CSS (Design via pencil.dev MCP)
- **Backend:** Node.js (Express)
- **AI Core:** LLM + RAG (Makestar Context Injection)
- **Intelligence:** Claude Code (Agentic Development)

## 📐 개발 원칙 (Ultrawork / Ralplan / Jogyo / Gyosu)
1. **Ultrawork:** 몰입도 높은 집중 개발 세션을 통해 핵심 기능을 빠르게 구현합니다.
2. **Ralplan:** 철저한 선행 설계와 구조화된 마일스톤을 준수합니다.
3. **Jogyo:** 보조 에이전트(Jogyo)를 활용해 반복적인 테스트 및 문서화를 자동화합니다.
4. **Gyosu:** 기술적 의사결정 시 깊이 있는 분석과 최선의 아키텍처를 지향합니다.
